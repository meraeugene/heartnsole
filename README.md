# HeartnSole.ph Landing Page

A premium, feminine-luxury sneaker boutique landing page for HeartnSole.ph, built as a standalone HTML website with Tailwind CSS, Google Fonts, Lucide icons, and AOS scroll animations.

## Features

- Premium blush pink, rose red, cream, black, and subtle green brand palette
- Responsive landing page layout
- Hero sneaker showcase with animated product visuals
- Featured sneaker collection cards
- Why HeartnSole brand trust section
- Payment options section
- Store hours and location sections
- Customer reviews section
- Minimal footer with contact and social links

## Tech Stack

- HTML
- Tailwind CSS CDN
- Google Fonts
- Lucide Icons
- AOS Animation Library
- Vanilla JavaScript

## Preview

Open `index.html` directly in a browser.
